Privacy Policy

ChannelEmotes Twitch Extension  ·  Last updated: June 16, 2026

Overview

ChannelEmotes is a Twitch Panel and Video Overlay Extension that displays the emotes available in a broadcaster's channel, sourced from 7TV and Twitch. This policy explains what information the Extension accesses, how it is used, and what is not collected.

Information We Access

The Extension accesses the following information provided by the Twitch Extension SDK:

  • Channel ID – A numeric identifier for the broadcaster's channel. This is used solely to query the 7TV API (7tv.io) and the Twitch Helix API for that channel's active emotes.
  • Extension Helix Token – A short-lived, read-only token provided by Twitch to the Extension. It is used exclusively to make authorised requests to Twitch's emote endpoints (/helix/chat/emotes and /helix/chat/emotes/global). The token is never stored or transmitted to any third-party server.

Information We Do Not Collect

  • We do not collect, store, or transmit any personally identifiable information (PII).
  • We do not use Identity Linking (ID Share) or request viewer identity information.
  • We do not use cookies, local storage, or any other persistent storage mechanism.
  • We do not integrate any analytics, advertising, or tracking services.
  • We do not log IP addresses or user behaviour.

Third-Party Services

The Extension communicates with the following external services to retrieve emote data:

  • 7TV API (7tv.io) – Requests are limited to fetching publicly available emote sets for the current channel. Please refer to 7TV's Privacy Policy for information on how 7TV handles requests to its API.
  • Twitch Helix API (dev.twitch.tv) – Used to retrieve the broadcaster's Twitch emotes and global Twitch emotes. Please refer to Twitch's Privacy Notice for information on how Twitch handles API requests.

Data Retention

The Extension does not retain any data. All emote data is fetched on demand when the Extension loads and exists only in memory for the duration of the current browser session. It is discarded when the page is closed or refreshed.

Children's Privacy

The Extension does not knowingly collect any information from users of any age, including children under the age of 13.

Changes to This Policy

If this policy is updated, the revised version will be published at this URL with an updated "Last updated" date. Continued use of the Extension after changes are posted constitutes acceptance of the revised policy.

Contact

If you have any questions about this privacy policy, please open an issue on the project's GitHub repository.
